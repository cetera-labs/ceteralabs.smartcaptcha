# Модуль Bitrix - ceteralabs.smartcaptcha

## Описание
Модуль автоматически заменяет обычную капчу 1С-Битрикс на [Yandex SmartCaptcha](https://yandex.cloud/ru/services/smartcaptcha).
* Поддержка всех форм сайта: регистрация, авторизация, обратная связь;
* Поддержка нескольких форм на одной странице;
* Поддержка ajax-форм;

__Если на странице отсутствуют формы с капчей, дополнительные скрипты не будут подключаться!__

### Получение ключей:
Способы получения ключей можно узнать в документации по [ссылке](https://yandex.cloud/en/docs/smartcaptcha/operations/get-keys).

### Настройка модуля:
В административном разделе  ( Настройки > Настройки продукта > Настройки модулей > CeteraLabs SmartCaptcha ) вставить полученные ключи.

### Дополнительно:
В настройках можно изменить подпись и стандартное сообщение об ошибке.
